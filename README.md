# 🍹 Aperitivo

> A sophisticated, dark-mode color palette inspired by warm peach orchards and chilled evening spirits.

**Aperitivo** is a universal theme designed to replace cold, blue-light-heavy dark modes with a cozy, high-contrast experience. Built on a foundation of **Deep Charcoal** and **Juicy Peach**, it strikes a balance between professional clarity and late-night comfort.

## ✨ Features

* **Warm Tones:** Reduced eye strain by focusing on the orange/red spectrum.
* **High Scannability:** Semantic coloring that differentiates logic, data, and metadata at a glance.
* **Universal Support:** Designed to be ported to VS Code, Neovim, Terminal Emulators, and more.

## 🚀 Installation

### VS Code
Currently, you can apply **Aperitivo** by adding the color definitions to your `settings.json` under `workbench.colorCustomizations`. A full `.vsix` extension is in the works!

### XAML / Windows
If you are using the Aperitivo XAML resources:
1. Copy the `SolidColorBrush` definitions into your App Resources.
2. Reference the keys (e.g., `StaticResource Base`) in your UI elements.
