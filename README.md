# 🍹 Aperitivo

> A warm, dark theme inspired by sun-drenched peach orchards and late-evening spirits. 

**Aperitivo** is a high-contrast, cozy dark theme designed for long coding sessions. It replaces the cold blues of traditional dark modes with a sophisticated palette of **Juicy Peach**, **Deep Charcoal**, and **Sage Green**.

## 🎨 Palette

| Color | Hex | Usage |
| :--- | :--- | :--- |
| `#1e1a1b` | **Base** | Main Background |
| `#ff9e64` | **Juicy Peach** | Keywords & Primary Accents |
| `#f6c177` | **Apricot** | Strings & Constants |
| `#a3be8c` | **Sage** | Functions & Success |
| `#eaddcf` | **Cream** | Primary Foreground Text |

## ✨ Features

* **Easy on the Eyes:** Low-blue-light palette reduces eye strain during night shifts.
* **Semantic Contrast:** Distinct colors for logic, types, and variables to improve code readability.
* **Universal Vibe:** Designed to look consistent across VS Code, Vim, Terminal, and your desktop.
